1. START YOUR LOCAL SERVER

2. OPEN IN BROWSER
   Navigate to one of these URLs (depending on your setup):

   MAMP USERS (DocumentRoot = /Users/ajaynicolas/Documents/GitHub/IBL5):
   • http://localhost/ibl5/demo.php ✓ CORRECT
   • http://localhost:8888/ibl5/demo.php (if using port 8888)

   • http://localhost:8000/demo.php

   WRONG URLs (will not work):

   • http://localhost/ibl5/demo-debug.php

================================================================================
TROUBLESHOOTING
================================================================================

CSS NOT LOADING?
  Solution: npm run build:css

MOBILE MENU EMPTY?
  Cause: blocks/ directory not accessible
  Solution: Check blocks/ exists and contains .php files

ALPINE.JS ERRORS?
  Cause: node_modules not installed
  Solution: npm install

PAGE ERRORS?
  Check: PHP error logs
  Verify: mainfile.php loads correctly
  Test: Classes autoload from classes/ directory
